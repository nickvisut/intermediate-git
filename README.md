# Intermediate Git, Part 1

## Setup

### Step 1: Fork this repo

first, fork this repo (https://github.com/nickvisut/intermediate-git)

### Step 2: Clone your fork

```shell
git clone https://github.com/your-username/intermediate-git.git
```

### Step 3: Switch to fix branch

```shell
git switch -c fix/sum
```

### Step 4: Create a new branch so that we can keep `fix/sum` intact

```shell
git switch -c fix/sum-func
```
